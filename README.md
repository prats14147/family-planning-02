## Authentication

This app supports:

- Username/password (hashed with Werkzeug)
- Google Sign-In (OIDC via Authlib)

### Setup

1. Install dependencies:
   - With uv/pip: `pip install -r <generated by pyproject>`, or use your toolchain to install from `pyproject.toml`.
2. Create Google OAuth credentials:
   - Create a Web application in Google Cloud Console.
   - Authorized redirect URI: `http://localhost:5000/auth/google`
3. Set environment variables:
   - `GOOGLE_CLIENT_ID`
   - `GOOGLE_CLIENT_SECRET`
4. Run the app:
   - `python app.py`

### Usage

- Local auth: Sign up, then log in.
- Google: Click "Continue with Google" on login or signup screens.
